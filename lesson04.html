การลงโปรแกรมพื้นฐานที่ควรมี

โปรแกรมสำคัญที่จำเป็นต่อการใช้งานคอมพิวเตอร์

หน้าแรก บทเรียน ประวัติคอมพิวเตอร์ เกี่ยวกับผู้จัดทำ

โปรแกรมระบบและความปลอดภัย

  • ระบบปฏิบัติการ (Operating System): Windows / macOS / Linux
  • โปรแกรมป้องกันไวรัส: เช่น Windows Defender (ติดมากับ Windows), Avast, Bitdefender
  • อัปเดตระบบ: ช่วยอุดช่องโหว่และเพิ่มความเสถียร

โปรแกรมใช้งานเอกสาร (สำนักงาน)

  • Microsoft Office: Word, Excel, PowerPoint
  • Google Docs / Sheets / Slides (ออนไลน์)
  • LibreOffice (ฟรี ใช้แทน Office ได้)

โปรแกรมอินเทอร์เน็ตและสื่อสาร

  • เว็บเบราว์เซอร์: Google Chrome, Microsoft Edge, Firefox
  • สื่อสาร/ประชุมออนไลน์: LINE, Zoom, Microsoft Teams

โปรแกรมอ่านไฟล์และบีบอัดไฟล์

  • อ่านไฟล์ PDF: Adobe Acrobat Reader
  • บีบอัด/แตกไฟล์: WinRAR, 7-Zip

โปรแกรมมัลติมีเดีย

  • เล่นวิดีโอ/เพลง: VLC Media Player, Windows Media Player

โปรแกรมตกแต่งภาพ/ออกแบบ (พื้นฐาน)

  • ตกแต่งภาพ: Adobe Photoshop
  • ออกแบบง่าย ๆ: Canva
  • ทางเลือกฟรี: GIMP

โปรแกรมเสริมที่ควรมี

  • จัดการไฟล์/โน้ต/เขียนโค้ด: Notepad++, Visual Studio Code
  • สำรองข้อมูล (Cloud): Google Drive, OneDrive, Dropbox
