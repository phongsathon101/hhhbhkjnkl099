การติดตั้งวินโดว์ 10และ11ด้วย USB

ขั้นตอนการลงระบบปฏิบัติการอย่างถูกต้อง

หน้าแรก บทเรียน ประวัติคอมพิวเตอร์ เกี่ยวกับผู้จัดทำ

ต่อไปนี้คือ ขั้นตอนการติดตั้ง Windows 10 และ Windows 11 อย่างถูกต้อง เหมาะสำหรับการเรียน การสอน และการใช้งานจริง แบ่งเป็นลำดับเข้าใจง่าย

1.เตรียมความพร้อมก่อนติดตั้ง

  • เตรียม แฟลชไดรฟ์อย่างน้อย 8 GB
  • สำรองข้อมูลสำคัญ (ไฟล์ รูป งาน) ไปยังแฟลชไดรฟ์หรือคลาวด์
  • ตรวจสอบสเปกเครื่อง
  • Windows 10: CPU 1 GHz, RAM 2 GB (64-bit แนะนำ 4 GB+)
  • Windows 11: ต้องมี TPM 2.0, Secure Boot, CPU รุ่นที่รองรับ

2.สร้างแฟลชไดรฟ์ติดตั้ง Windows (Boot USB)

  • ดาวน์โหลด Media Creation Tool (จาก Microsoft)
  • เสียบแฟลชไดรฟ์ → เปิดโปรแกรม
  • เลือก Create installation media
  • เลือกเวอร์ชัน Windows 10 หรือ 11
  • เลือก USB Flash Drive
  • รอจนสร้างเสร็จ

3.ตั้งค่า BIOS / UEFI ให้บูตจาก USB

  • รีสตาร์ตเครื่อง
  • กดปุ่มเข้า BIOS/Boot Menu (เช่น F2 / F10 / F12 / DEL)
  • ตั้งค่า Boot Order ให้ USB เป็นลำดับแรก
  • Save & Exit

4.เริ่มติดตั้ง Windows

  • เลือกภาษา เวลา และคีย์บอร์ด
  • คลิก Install Now
  • ใส่ Product Key (หรือเลือก I don’t have a product key)
  • เลือกรุ่น Windows ให้ตรงกับไลเซนส์

5.เลือกประเภทการติดตั้ง

  • เลือก Custom: Install Windows only (advanced)
  • ลบ/สร้างพาร์ทิชัน (กรณีลงใหม่ทั้งเครื่อง)
  • เลือกไดรฟ์หลัก → กด Next
    • ⚠️ การลบพาร์ทิชันจะทำให้ข้อมูลหาย

6.รอระบบติดตั้งอัตโนมัติ

  • เครื่องจะคัดลอกไฟล์และรีสตาร์ตอัตโนมัติ
  • ห้ามถอด USB จนกว่าจะเข้าหน้าตั้งค่าแรกของ Windows

7.ตั้งค่าเริ่มต้นหลังติดตั้ง

  • เลือกประเทศ ภาษา คีย์บอร์ด
  • เชื่อมต่ออินเทอร์เน็ต
  • สร้างบัญชีผู้ใช้ (Microsoft Account หรือ Local Account)
  • ตั้งค่าความเป็นส่วนตัว

8.หลังติดตั้งเสร็จ (สำคัญมาก)

  • อัปเดต Windows Update ให้ครบ
  • ติดตั้ง ไดรเวอร์ (การ์ดจอ เสียง Wi-Fi)
  • ติดตั้งโปรแกรมพื้นฐาน เช่น Office, Browser, PDF Reader
  • เปิดใช้งาน Windows (Activate)
